### Projects

#### ⚡ ZeevCode — Online Coding Platform
> Solo-built · 20+ problems · 1v1 real-time duels · **[Live](https://zeevcode.vercel.app/)** · **[GitHub](https://github.com/sarzeev/zeevcode)**

The headline: I wrote my own code execution engine. `ProcessBuilder` sandbox in Java — compiles and runs untrusted user code with per-test-case time and memory enforcement. No Judge0, no third-party API.

- AWS SQS job queue decoupling the API layer from execution workers — async processing, horizontal scaling under load
- Real-time 1v1 duel mode over WebSockets — live sync of problem state, countdown timers, and submission results across both players simultaneously
- Spring Boot + PostgreSQL (MVC), Firebase Auth with role-based admin access, Nginx on EC2, React.js frontend on Vercel

`Java` `Spring Boot` `PostgreSQL` `React.js` `AWS EC2` `SQS` `WebSockets` `Firebase Auth` `Nginx`
